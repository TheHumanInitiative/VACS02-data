# VACS02-data
It's believed that vacancy data can potentially be used as an indicator as to how the labour market is being effected by AI. Here we use data collected by the ONS (VACS 02) to analyse how different industry sectors are being effected.

Select the title of each industry in the legend to disable On/Off

VACS 02 data is available here https://www.ons.gov.uk/employmentandlabourmarket/peoplenotinwork/unemployment/datasets/vacanciesbyindustryvacs02
